# 🏨 TripAdvisor Review Sentiment Analysis (NLP Project)

## 📌 Project Overview

This project applies **Natural Language Processing (NLP)** techniques to analyze customer reviews from TripAdvisor. The objective is to extract meaningful insights from unstructured text data and identify patterns in customer sentiment.

By converting textual reviews into structured numerical representations, the project helps understand customer satisfaction drivers and areas for improvement.

---

## 🎯 Objectives

- Perform text preprocessing and cleaning
- Transform textual data into machine-readable format
- Apply feature extraction techniques (TF-IDF)
- Build sentiment classification models
- Evaluate model performance
- Generate actionable business insights

---

## 📂 Dataset Information

- **File Name:** `Trip_advisor_review.csv`
- Contains customer hotel reviews
- Each row represents one review
- Includes review text and rating information

---

## 🛠️ Technologies Used

- Python

- Matplotlib / Seaborn
- NLTK
- Scikit-learn

## 🔄 Project Workflow

### 1️⃣ Data Loading
- Imported TripAdvisor review dataset
- Checked dataset structure and missing values

### 2️⃣ Text Preprocessing
- Converted text to lowercase
- Removed punctuation and special characters
- Removed stopwords
- Tokenization
- Lemmatization

### 3️⃣ Feature Engineering
- Applied TF-IDF Vectorization
- Converted textual reviews into numerical format

### 4️⃣ Model Building
- Trained classification models
- Predicted sentiment labels

### 5️⃣ Model Evaluation
- Accuracy Score
- Confusion Matrix
- Classification Report

## ⭐ 1. Sentiment Distribution
- Majority of reviews were positive.
- Negative reviews were fewer but more detailed.

## ⭐ 2. Positive Experience Drivers
Frequent positive keywords:
- Great location
- Friendly staff
- Clean rooms
- Comfortable stay

## ⭐ 3. Negative Experience Drivers
Common negative themes:
- Poor service
- Unclean rooms
- Noise issues
- Delays

## ⭐ 4. Review Behavior Pattern
- Positive reviews tend to be shorter and emotionally expressive.
- Negative reviews are generally longer and more descriptive.

## ⭐ 5. Business Insight
- Service quality and cleanliness strongly influence customer satisfaction.
- Location advantage significantly impacts positive ratings.
- Operational improvements in service response time can reduce negative feedback.
